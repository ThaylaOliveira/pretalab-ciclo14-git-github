# Instalação e ativação da chave SSH

1. Verificar se já existe a chave:
ls ~/.ssh

2. Instalar uma nova chave:
ssh-keygen -t ed25519 -C "seu-email@exemplo.com"

3. Ativar o agente SSH e carregar a chave:
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

4. Exibir as chaves para copiar:
// Pública: cat ~/.ssh/id_ed25519.pub
// Privada: cat ~/.ssh/id_ed25519
! Exemplo de chave pública: ssh-ed25519 ...Z1YYm+5K4 seu-email@mail.com

5. Ativar no Github
- Copiar a chave pública para o GitHub:
- Acesse GitHub → Settings → SSH and GPG keys → New SSH key
- Cole a chave pública e salve.
